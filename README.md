# AI-Emotional-Support-Companion-System
- The AI Emotional Support Companion System is a web-based application that provides emotional support using an AI chatbot. It allows users to communicate through text or voice, where the system understands emotions and responds with empathetic and meaningful conversations.
- The system also suggests helpful actions such as relaxation techniques and coping strategies to improve user well-being.

## Features
-  Real-time AI chatbot conversation
- Voice input and output support
- Emotion-aware responses
- Mood tracking (optional)
- Suggestion system (coping tips, activities)
- Cloud-based data storage

## Technologies Used
#### Backend
- Python
- FastAPI / Flask
#### AI / ML
- TinyLlama (Fine-tuned model)
- PyTorch
- Hugging Face Transformers
#### Frontend
- HTML, CSS, Bootstrap / React
#### Database
- MongoDB

#### Additional Tools
- Whisper (Speech-to-Text)
- gTTS / Coqui TTS (Text-to-Speech)

## How It Works
1. User enters text or voice input  
2. Backend processes the input  
3. AI model analyzes emotional context  
4. Model generates response  
5. Suggestions are provided  
6. Output is displayed to the user
